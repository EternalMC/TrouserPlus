# TrouserPlus (Meteor Client Addon)

**TrouserPlus** is a powerful, feature-rich addon for the **Meteor Client**, specifically designed for the latest Minecraft Anarchy environments. It bridges the gap between classic utility and modern combat, integrating the legendary **TrouserStreak** with a suite of exclusive, high-performance modules.

---

## ✨ Key Features

### ⚔️ Combat & PvP (The Powerhouse)

* **B Mace Reach (Exclusive!)**: My signature module designed for the 1.21+ Mace meta. It optimizes attack reach and hit detection, ensuring your smash attacks land with pinpoint accuracy even from extreme heights.
* **UltraAura**: A high-speed, multi-target KillAura with advanced prediction logic that outperforms standard combat modules.
* **TotemBypass**: Engineered to bypass specific anti-cheat checks, ensuring your crystals or hits land effectively even against geared opponents.
* **NoCooldownDamage**: Maximum DPS. Removes the attack delay constraints to shred enemies instantly.
* **Infinite Reach**: Interact with objects and attack entities from distances that shouldn't be possible.

### 🗺️ Utility & Exploration

* **MassTP**: Efficiently teleport to specific coordinates or entities across the map.
* **Full TrouserStreak Integration**: Includes all the classic tools like NewChunks (for finding old bases), HoleESP, and more, but with improved stability.
* **Java & Bedrock Optimized**: Fine-tuned to work flawlessly on cross-play servers like FinalAnarchy.

---

## 🚀 Installation

1. **Requirements**: Ensure you have **Fabric Loader** and the latest version of **Meteor Client** installed.
2. **Download**: Go to the [Releases](https://github.com/Tocfyr/TrouserPlus/releases) page and download the latest `.jar` file.
3. **Install**: Drop the `.jar` file into your Minecraft `%appdata%/.minecraft/mods` folder.
4. **Launch**: Start your game and find the **TrouserPlus** tab in the Meteor Client click GUI.
